# Sanic Template API

## Setup environment

* Install requirements:
  > pip3 install -r requirements.txt

* Environment file (.env):
    * Example at: [.env.example](.env.example)
     
## Run product

> docker-compose up

## Run dev

### Run Rest API Server
```
python3 main.py
```
